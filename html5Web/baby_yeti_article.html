Robot testbench for software development

Overview
This project was for my senior design class. For anyone not knowledgable in the ways of senior design, it is a two-part class which takes groups of students along the engineering design process.
My group went with a small robot which would be a software testing platform. More specifically, a software testing platform for the Intelligent Systems Club's autonomous snowplow team "Yeti"; now known as "Mammoth". I talk about the Yeti project Here. This project solves an issue that the team was having, where they did not have enough testing time leading up to the 2020 competition. The testbench provides a tool to work with when the mechanical and electrical teams have the Yeti robot in pieces. The testbench also provides a way for new members to experiment with their code outside of Gazebo and the likes.
