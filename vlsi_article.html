VLSI Chip Design

Overview
This is a final project from an introductary VLSI course. It was built using the L-Edit 2016.2 software.
For the assignment, 0.5-micron technology is one of the design requirements. The lambda is equal to 5e-7m. Portions of the Tanner library in L-Edit were used as building blocks, such as the frame pads. Previous assignments from the class were used as building blocks as well. They were then cascaded and added together. The final design features a static-resistant frame, a four-bit ALU, and an output register.

Design
The overall design for the four-bit ALU comes from the cascaded version of a one-bit ALU. The basic design is shown in Figure 1. The ALU provides functions and will produce outputs based on which selection for the function is selected. The functions selected for this project are: Add/Sub, XOR, AND, OR. A register system is added with the purpose of demonstrating a sequential design.

First, I will look at the indivisual components starting with the ALU. The truth tables are created first and then the Karnaugh maps produce the equations. The Minimized equation is shown in Figure 2. For the sake of efficient design, the majority of gates considered for this project are taken as constraints of NAND and NOT gates. Some extra gates such as XOR gates are also included, but the logic generated for most of the final components are reduced to NAND and NOT gates. This is to minimize the physical area that the layout will occupy. These two gates require very little space in a chip layout. The minimized equations contain a conversion to a majority of NAND and NOT gates.

The MUX component will serve as a way to select which logical function will be produced from the ALU. The MUX for this design will need to take in four inputs and produce one output. Two select lines are needed for this functionality. In order to use the 2x1 MUX that is in the standard cells library, three MUXs must be combined to create the 4x1 MUX. The anti-static frame is considered next. The frame is created with the components for the Tanner anti-static library. The number of inputs and outputs must be determined to create the chip design. Reset and clock are taken as general inputs. Two select lines are needed for each bit of ALU, so four S1 and four S0 inputs are added. A and B are needed as inputs for each ALU bit. Cin is needed for just the first ALU bit. four A, four B, and one Cin inputs are added. A Cout bit is needed for the last ALU bit. One output bit is needed for each of the four ALU bits. This brings the total of twenty-six input or output pads. Two spacer pads are added simply to create a square design. Four sides of seven pads are used.
